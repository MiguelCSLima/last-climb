# Last Climb

Last Climb é um jogo de plataforma 2D desenvolvido no GDevelop 5 como projeto acadêmico.

## Sobre o jogo

O jogo apresenta uma progressão vertical, na qual o jogador precisa superar obstáculos, inimigos e desafios para chegar ao topo. A proposta combina exploração, movimentação em plataforma, fases com dificuldade progressiva e uma batalha final contra chefe.

## Tecnologias utilizadas

- GDevelop 5
- HTML5/JavaScript na exportação web
- Assets 2D em pixel art

## Funcionalidades

- Personagem jogável com movimentação em plataforma
- Fases com obstáculos
- Inimigos
- Cutscenes
- Sistema de dano
- Sons e música de fundo
- Tela de créditos
- Boss final
